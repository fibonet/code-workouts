# Problem 4: Mini Maze Solver

## Task
Given a 2D grid with:
- `0 = empty`
- `1 = wall`

Determine if there is a path from top-left to bottom-right using only **down** or **right** moves.

## Examples
Input:  
```
[[0,0,1],
 [0,0,0],
 [1,0,0]]
```
Output: `True`

Input:  
```
[[0,1],
 [1,0]]
```
Output: `False`
